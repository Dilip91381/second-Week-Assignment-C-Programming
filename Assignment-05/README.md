# 💰 Bank Interest Calculator (C)

A simple C program that calculates **Simple Interest** based on the principal amount, annual interest rate, and time period entered by the user.

## 📌 Features

- Accepts Principal Amount from the user
- Accepts Rate of Interest (%)
- Accepts Time (in Years)
- Calculates Simple Interest
- Displays the result in a clean format

## 🛠️ Technologies Used

- C Programming Language
- Standard Input/Output Library (`stdio.h`)

## 📂 Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## ▶️ How to Run

1. Save the program as `bank_interest_calculator.c`
2. Compile the program:

```bash
gcc bank_interest_calculator.c -o bank_interest_calculator
```

3. Run the executable:

```bash
./bank_interest_calculator
```

**For Windows:**

```bash
bank_interest_calculator.exe
```

## 💻 Sample Input

```
Enter Principal Amount: 10000
Enter Rate of Interest: 7.5
Enter Time (Years): 2
```

## 📤 Sample Output

```
----- Bank Interest -----
Principal : 10000.00
Rate      : 7.50%
Time      : 2.00 Years
Interest  : 1500.00
```
