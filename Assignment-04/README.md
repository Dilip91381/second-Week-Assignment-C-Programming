# Currency Exchange System

## 📌 Description

The **Currency Exchange System** is a simple C program that converts an amount from **Indian Rupees (INR)** to **United States Dollars (USD)** using a predefined exchange rate.

The program accepts an amount in INR from the user, performs the currency conversion, and displays the equivalent amount in USD.

> **Exchange Rate Used:**  
> **1 USD = ₹96.36**

---

## 🎯 Objective

- Accept an amount in Indian Rupees (INR).
- Convert the amount into US Dollars (USD).
- Display the converted amount with proper formatting.

---
