# Student Performance Analyzer

## 📌 Description

The **Student Performance Analyzer** is a simple C program that accepts marks for **five subjects** from the user and calculates the following:

- Total Marks
- Average Marks
- Percentage

This project helps beginners understand user input, arithmetic operations, variables, and formatted output in C programming.

---

## 🎯 Objective

To develop a program that:
- Accepts marks of five subjects.
- Calculates the total marks obtained.
- Computes the average marks.
- Determines the percentage.
- Displays the result in a clear and formatted manner.

---

## 💻 Concepts Used

- Variables
- Input/Output (`scanf`, `printf`)
- Arithmetic Operators
- Mathematical Calculations
- Formatted Output

## 🧮 Formula Used

```
Total = m1 + m2 + m3 + m4 + m5

Average = Total / 5

Percentage = Average
```

> **Note:** Since the total marks are assumed to be **500**, the percentage is equal to the average.
