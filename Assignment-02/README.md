# Employee Salary Processing

## 📌 Description

This C program calculates an employee's monthly salary by accepting the Basic Salary and computing:

- HRA (20%)
- DA (40%)
- Net Salary

## 💻 Concepts Used

- Variables
- Input/Output
- Arithmetic Operators

## 📥 Sample Input

```
Basic Salary = 30000
```

## 📤 Sample Output

```
HRA = 6000
DA = 12000
Net Salary = 48000
```

## ▶️ How to Run

Compile the program:

```bash
gcc employee_salary_processing.c -o salary
```

Run the program:

```bash
./salary
```

For Windows:

```bash
salary.exe
```

## 📚 Learning Outcome

- Accept user input
- Perform arithmetic calculations
- Display formatted output
